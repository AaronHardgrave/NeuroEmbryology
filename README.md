## Neuroembryology Learning Module

This repository contains the Neuroembryology Learning Module with interactive 3D visualization.

### View the Module Online
[Click here to view the module](https://aaronhardgrave.github.io/NeuroEmbryology/)

### How to Enable GitHub Pages
1. Go to **Settings** > **Pages**.
2. Under **Source**, select `main` branch and `/root`.
3. Save. Your site will be live at:
   `https://aaronhardgrave.github.io/NeuroEmbryology/`

### Contents
- `index.html` – Full interactive module with Sketchfab embed and figure placeholders.
- `images/` – Add your figure images here.
